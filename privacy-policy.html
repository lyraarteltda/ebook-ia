Política de Privacidade

Última atualização: 2026-06-20

Esta Política de Privacidade descreve como a ferramenta E-book IA, oferecida pela comunidade Maestros da IA, coleta, utiliza e protege os dados pessoais dos seus usuários, em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018).

1. Dados Coletados

Coletamos apenas os dados estritamente necessários para o funcionamento da ferramenta:

  • E-mail: para verificação de acesso à comunidade.
  • Telefone (WhatsApp): para verificação de acesso à comunidade.
  • Feedback (opcional): a mensagem que você nos envia voluntariamente pela ferramenta.

Sua chave de API de terceiros (OpenRouter, OpenAI) é armazenada apenas no navegador (localStorage) e nunca é transmitida para nossos servidores. O conteúdo dos e-books que você gera é processado diretamente entre o seu navegador e o provedor de IA escolhido — nós nunca o recebemos nem o armazenamos.

2. Finalidade do Tratamento

  • Verificação de acesso: confirmar que o usuário é membro ativo da comunidade.
  • Suporte e melhoria: responder ao feedback enviado voluntariamente.

3. Base Legal

O tratamento de dados é realizado com base no legítimo interesse (Art. 7º, IX da LGPD) para verificação de acesso e na execução de contrato (Art. 7º, V) referente à assinatura da comunidade Maestros da IA.

4. Compartilhamento de Dados

Seus dados não são vendidos ou compartilhados com terceiros para fins comerciais. Utilizamos os seguintes serviços técnicos:

  • Supabase: banco de dados para verificação de membros, acessado somente pelo nosso servidor (n8n) — o seu navegador nunca acessa o banco diretamente.
  • n8n: camada de servidor que executa a verificação de acesso e recebe o feedback.
  • Netlify: hospedagem da aplicação web (site estático).
  • Provedor de IA escolhido (OpenRouter / OpenAI): recebe diretamente do seu navegador o conteúdo que você gera, usando a sua própria chave.

5. Período de Retenção

  • Dados de verificação: mantidos enquanto o usuário for membro ativo da comunidade.
  • Mensagens de feedback: mantidas pelo tempo necessário ao atendimento.

6. Seus Direitos (LGPD)

Você tem direito a:

  • Confirmação da existência de tratamento de dados;
  • Acesso aos seus dados pessoais;
  • Correção de dados incompletos ou desatualizados;
  • Eliminação dos dados pessoais (veja o link abaixo);
  • Informação sobre compartilhamento de dados;
  • Revogação do consentimento.

7. Segurança

Implementamos medidas técnicas para proteger seus dados:

  • Comunicação criptografada (HTTPS/TLS);
  • Row-Level Security (RLS) no banco de dados;
  • Verificação de acesso exclusivamente no servidor (o navegador não toca o banco);
  • Limitação de taxa de requisições;
  • Chaves de API nunca armazenadas no servidor.
